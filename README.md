# SmartClass 🎓

**Мобильное приложение для изучения алгебры (7–9 классы) с ролевой моделью доступа.**

Дипломный проект, разработанный в рамках специальности **09.02.07 — Информационные системы и программирование**.

## 📱 О проекте

**SmartClass** — это образовательное мобильное приложение на Android, предназначенное для повышения эффективности изучения алгебры. Приложение решает проблему разрозненности учебных материалов, объединяя теорию, практику и систему контроля знаний в едином интерфейсе.

### Ключевые особенности:
*   📚 **Структурированная теория:** Материал разбит по темам для 7, 8 и 9 классов.
*   📝 **Интерактивное тестирование:** Мгновенная проверка знаний с сохранением результатов.
*   👥 **Ролевая модель:** Раздельные интерфейсы для **Ученика** и **Учителя**.
*   ☁️ **Облачная синхронизация:** Данные хранятся в Firebase Firestore.
*   🛡️ **Безопасность:** Авторизация через Firebase Auth и защита данных.

Проект разработан с использованием современных инструментов экосистемы Android:

## ⚙️ Установка и Запуск

### Требования
*   Android Studio Hedgehog | 2023.1.1 или новее
*   JDK 17+
*   Аккаунт Firebase (для работы бэкенда)

### Инструкция
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/marybetc/SmartClass.git
    cd SmartClass
    ```
2.  Откройте проект в **Android Studio**.
3.  Создайте проект в [Firebase Console](https://console.firebase.google.com/).
4.  Скачайте файл `google-services.json` и поместите его в папку `app/`.
5.  Включите в Firebase:
    *   **Authentication:** Email/Password provider.
    *   **Firestore Database:** Создайте базу данных (начните в тестовом режиме или настройте правила).
6.  Запустите приложение на эмуляторе или физическом устройстве.

## 🏗 Структура базы данных (Firestore)

Приложение использует следующую структуру коллекций:

*   `users`
    *   `uid`: { role: "student" | "teacher", name: "...", class: "7A" }
*   `topics`
    *   `topic_id`: { title: "Линейные уравнения", grade: 7, content: "...", formulas: [...] }
*   `tests`
    *   `test_id`: { topicId: "...", questions: [...], difficulty: "easy" }
*   `results`
    *   `result_id`: { studentId: "...", testId: "...", score: 90, date: timestamp }
